# Необхідні зображення для лендингу

Для повноцінного відображення лендингу потрібно додати наступні зображення в папку `public/assets/`:

## 1. `/public/assets/preview.png`
- **Призначення**: Головне зображення в Hero секції (HeroContainerScroll)
- **Розміри**: Рекомендовано 1200x1200px або більше
- **Опис**: Скріншот або демонстрація інтерфейсу GeoSim платформи
- **Контент**: Може показувати дашборд симуляції, графіки, або інтерфейс моделювання

## 2. `/public/assets/feature-extraction.png`
- **Призначення**: Зображення для секції "Extraction Kinetics Modeling" в StickyScroll
- **Розміри**: Рекомендовано мінімум 800x600px
- **Опис**: Візуалізація процесу екстракції кінетики
- **Контент**: Графіки екстракції, реакції, або моделювання процесів

## 3. `/public/assets/feature-hvp.png`
- **Призначення**: Зображення для секції "HVP Liberation Analysis" в StickyScroll
- **Розміри**: Рекомендовано мінімум 800x600px
- **Опис**: Візуалізація HVP (High Voltage Pulse) процесів
- **Контент**: Графіки HVP, енергетичні криві, або візуалізація процесу

## 4. `/public/assets/feature-visualization.png`
- **Призначення**: Зображення для секції "Metallurgical Visualization" в StickyScroll
- **Розміри**: Рекомендовано мінімум 800x600px
- **Опис**: Приклади візуалізації металургійних даних
- **Контент**: 3D моделі, графіки реакцій, або інтерактивні візуалізації

## 5. `/public/assets/feature-scenarios.png`
- **Призначення**: Зображення для секції "Scenario Analysis & Optimization" в StickyScroll
- **Розміри**: Рекомендовано мінімум 800x600px
- **Опис**: Приклади аналізу сценаріїв та оптимізації
- **Контент**: Порівняльні графіки, таблиці даних, або інтерфейс аналізу

## Рекомендації

- Всі зображення повинні бути оптимізовані для веб (формат PNG або WebP)
- Рекомендується використовувати високоякісні зображення з хорошим контрастом
- Зображення повинні відповідати тематиці GeoSim та мінерального оброблення
- Для кращої продуктивності можна використовувати WebP формат з fallback на PNG

## Альтернативні варіанти

Якщо у вас немає готових зображень, можна:
1. Використати скріншоти з поточного інтерфейсу GeoSim
2. Створити мокапи або wireframes для демонстрації
3. Використати тематичні ілюстрації, пов'язані з мінеральним обробленням
